SRun3K Stupid
=============

简介
----
深澜网络认证第三方客户端，仅供学习使用。优势如下：

- 稳定不崩溃
- 允许网络共享
- 启动快速
- 界面简洁
- 不需要提升权限

使用纯Python编写，基于Python3+PySide，跨平台。

用法
----
1. 运行SRun3K.py可启动基于PySide的GUI：

        $ python3 SRun3K.py

1. core.py作为程序的核心，同时又可以作为一个命令行版单独运行，而且兼容Python 2和3：

        $ python3 core.py

        $ python core.py

1. 编译成可执行程序：

        $ python setup.py build
